## 订单模块API

### 1. 创建订单

**接口**：`POST /api/orders`

**描述**：创建新订单，支持多商品、收货地址、联系信息。

**请求体**：
```json
{
  "customerId": "string (必填)",
  "customerName": "string (必填)",
  "items": [
    {
      "productId": "string (必填)",
      "productName": "string (必填)",
      "skuCode": "string (必填)",
      "unitPrice": {
        "amount": "decimal (必填)",
        "currency": "string (必填，默认CNY)"
      },
      "quantity": "integer (必填，≥1)"
    }
  ],
  "totalAmount": {
    "amount": "decimal (必填)",
    "currency": "string (必填，默认CNY)"
  },
  "shippingAddress": {
    "province": "string (必填)",
    "city": "string (必填)",
    "district": "string (可选)",
    "detailAddress": "string (必填)",
    "postalCode": "string (可选)"
  },
  "contactInfo": {
    "contactName": "string (必填)",
    "contactPhone": "string (必填)",
    "contactEmail": "string (可选)"
  },
  "remark": "string (可选)"
}
```

**响应体**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "orderNo": "ORDABC1234567890",
    "customerId": "customer123",
    "customerName": "张三",
    "status": "CREATED",
    "totalAmount": {
      "amount": "1999.00",
      "currency": "CNY"
    },
    "currency": "CNY",
    "paymentMethod": null,
    "remark": "备注信息",
    "items": [
      {
        "productId": "product001",
        "productName": "测试商品",
        "skuCode": "SKU001",
        "unitPrice": {
          "amount": "99.50",
          "currency": "CNY"
        },
        "currency": "CNY",
        "quantity": 10,
        "subtotal": {
          "amount": "995.00",
          "currency": "CNY"
        }
      }
    ],
    "shippingAddress": {
      "province": "北京市",
      "city": "北京市",
      "district": "朝阳区",
      "detailAddress": "某某街道123号",
      "postalCode": "100000"
    },
    "contactInfo": {
      "contactName": "张三",
      "contactPhone": "13800138000",
      "contactEmail": "zhangsan@example.com"
    },
    "createTime": "2026-01-28T07:00:00.000Z",
    "paymentTime": null,
    "shippedTime": null,
    "completedTime": null,
    "cancelledTime": null,
    "cancelReason": null
  },
  "timestamp": "2026-01-28T07:00:00.000Z"
}
```

**验证规则**：
- `customerId`: NotBlank - 客户ID不能为空
- `customerName`: NotBlank - 客户姓名不能为空
- `items`: NotEmpty - 订单项列表不能为空
- `items[].productId`: NotBlank - 商品ID不能为空
- `items[].productName`: NotBlank - 商品名称不能为空
- `items[].skuCode`: NotBlank - SKU编码不能为空
- `items[].unitPrice`: NotNull - 单价不能为空
- `items[].quantity`: 必须 ≥ 1 - 数量不能为空
- `totalAmount`: NotNull - 订单总金额不能为空
- `shippingAddress.province`: NotBlank - 省份不能为空
- `shippingAddress.city`: NotBlank - 城市不能为空
- `shippingAddress.detailAddress`: NotBlank - 详细地址不能为空
- `contactInfo.contactName`: NotBlank - 联系人姓名不能为空
- `contactInfo.contactPhone`: NotBlank - 联系电话不能为空

**业务规则**：
- 订单编号格式：ORD + 16位随机大写字母数字（如ORDABC1234567890）
- 订单状态转换：CREATED → PAID → SHIPPED → COMPLETED，各状态可CANCELLED
- 订单创建时自动生成订单号
- 订单金额计算：订单总金额 = ∑ (订单项单价 × 数量）

### 2. 支付订单

**接口**：`POST /api/orders/{orderId}/pay`

**描述**：支付指定订单，验证支付金额与订单金额匹配。

**路径参数**：
- `orderId` (Long): 订单ID，必填

**请求体**：
```json
{
  "paymentMethod": "string (必填)",
  "paymentAmount": {
    "amount": "decimal (必填)",
    "currency": "string (必填，默认CNY)"
  }
}
```

**响应体**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "orderNo": "ORDABC1234567890",
    "status": "PAID",
    "totalAmount": {
      "amount": "1999.00",
      "currency": "CNY"
    },
    "currency": "CNY",
    "paymentMethod": "ALIPAY",
    "paymentTime": "2026-01-28T07:05:00.000Z",
    "remark": "备注信息",
    "items": [ ... ],
    "shippingAddress": { ... },
    "contactInfo": { ... },
    "createTime": "2026-01-28T07:00:00.000Z",
    "shippedTime": null,
    "completedTime": null,
    "cancelledTime": null,
    "cancelReason": null
  },
  "timestamp": "2026-01-28T07:05:00.000Z"
}
```

**验证规则**：
- `paymentMethod`: NotBlank - 支付方式不能为空（ALIPAY、WECHAT、STRIPE）

**业务规则**：
- 仅CREATED状态的订单可以支付
- 支付金额必须与订单总金额完全匹配
- 支付后状态转换为PAID
- 支付方式枚举：ALIPAY（支付宝）、WECHAT（微信）、STRIPE（Stripe支付）

### 3. 取消订单

**接口**：`POST /api/orders/{orderId}/cancel`

**描述**：取消指定订单，释放库存。

**路径参数**：
- `orderId` (Long): 订单ID，必填

**请求体**：
```json
{
  "reason": "string (必填)"
}
```

**响应体**：
```json
{
  "code": 200,
  "message": "success",
  "data": {
    "id": 1,
    "orderNo": "ORDABC1234567890",
    "status": "CANCELLED",
    "totalAmount": {
      "amount": "1999.00",
      "currency": "CNY"
    },
    "currency": "CNY",
    "paymentMethod": null,
    "remark": "备注信息",
    "cancelReason": "用户取消了订单",
    "cancelledTime": "2026-01-28T07:10:00.000Z",
    "items": [ ... ],
    "shippingAddress": { ... },
    "contactInfo": { ... },
    "createTime": "2026-01-28T07:00:00.000Z",
    "paymentTime": "2026-01-28T07:05:00.000Z",
    "shippedTime": null,
    "completedTime": null,
    "cancelReason": "用户取消了订单"
  },
  "timestamp": "2026-01-28T07:10:00.000Z"
}
```

**验证规则**：
- `reason`: NotBlank - 取消原因不能为空

**业务规则**：
- CREATED或PAID状态的订单可以取消
- 取消后状态转换为CANCELLED
- 取消时需要释放库存（通过OrderCancelledEventHandler处理）
