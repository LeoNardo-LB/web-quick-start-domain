# 5分钟快速上手

> **目标**：在5分钟内启动项目并了解基本使用

## 前置要求

✅ **JDK 25+**（项目使用JDK 25）
✅ **Maven 3.8+**
✅ **MySQL 8.0+**（可选，默认使用H2内存数据库）

---

## 第1步：克隆项目（30秒）

```bash
git clone <repository-url>
cd web-quick-start-domain
```

---

## 第2步：编译项目（1分钟）

```bash
mvn clean compile
```

**预期输出**：
```
[INFO] BUILD SUCCESS
[INFO] Total time: XX s
```

⚠️ **如果编译失败**：
- 检查JDK版本：`java -version` 应该是 25+
- 检查Maven版本：`mvn -v` 应该是 3.8+

---

## 第3步：运行测试（2分钟）

### 3.1 运行单元测试

```bash
mvn test
```

**预期输出**：
```
Tests run: 180, Failures: 0, Errors: 0, Skipped: 0
[INFO] BUILD SUCCESS
```

### 3.2 运行启动测试（验证Spring上下文）

```bash
mvn test -Dtest=ApplicationStartupTests -pl test
```

**预期输出**：
```
✅ Application context started successfully
✅ EsClient Bean successfully loaded: DisabledEsClientImpl
✅ SearchService Bean successfully loaded: SearchServiceImpl
```

✅ **如果测试全部通过，说明项目环境配置正确！**

---

## 第4步：启动应用（1分钟）

```bash
mvn spring-boot:run -pl start
```

**预期输出**：
```
Started Application in X.XXX seconds (JVM running for X.XXX)
```

应用将在 **http://localhost:8080** 启动

---

## 第5步：测试API（30秒）

### 5.1 查看订单列表

```bash
curl http://localhost:8080/api/orders
```

**预期响应**：
```json
{
  "code": 200,
  "data": [...],
  "message": "success",
  "time": "2026-01-16T..."
}
```

### 5.2 查看API文档

打开浏览器访问：**http://localhost:8080/swagger-ui.html**

## ❓ 常见问题

### Q1: 编译时提示"不支持类文件"
**A**: JDK版本不对，请使用JDK 25+
```bash
java -version  # 应该显示 25.x.x
```

### Q2: 启动时提示"Connection refused"
**A**: MySQL未启动或配置错误
- 项目默认使用H2内存数据库，不需要MySQL
- 如果要使用MySQL，配置`application-integration.yml`

### Q3: 测试失败怎么办？
**A**: 按以下顺序排查
1. 清理：`mvn clean`
2. 重新编译：`mvn clean compile`
3. 运行单个测试：`mvn test -Dtest=xxx`
4. 查看错误日志

### Q4: 如何查看更多文档？
**A**:
- 项目概览：`../README.md`（项目根目录）
- 完整文档目录：`../CLAUDE.md`（项目根目录）
- AI开发指南：`../CLAUDE.md`（项目根目录）
